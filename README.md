# sacmat-transactions-sok-artifacts
Artifacts for SoK: Evolution, Security, and Fundamental Properties of Transactional Systems

artifact_paper_list.ris is an RIS (Research Information Systems) file which can be imported to a reference-management system (eg, Zotero). It contains reference details for 163 papers examined for the paper "SoK: Evolution, Security, and Fundamental Properties of Transactional Systems," published at ACM SACMAT 2026.
