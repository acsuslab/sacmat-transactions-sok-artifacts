# sacmat-transactions-sok-artifacts
The followig artifacts for "SoK: Evolution, Security, and Fundamental Properties of Transactional Systems" can be found at https://github.com/acsuslab/sacmat-transactions-sok-artifacts/


all_163_papers.ris is an RIS (Research Information Systems) file which can be imported to a reference-management system (eg, Zotero). It contains reference details for the 163 in-scope papers examined for the paper "SoK: Evolution, Security, and Fundamental Properties of Transactional Systems," published at ACM SACMAT 2026.

curated_set_of_41_papers.ris contains bibliographic information for the 41 papers presented in Table 2 of the paper.
